📄 一、前端页面
7 个核心页面

📝post.html · 帖子详情

📢notice.html · 通知发布者页

🛠️admin.html · 管理员后台

  • 敏感词 & 白名单、霸凌姓名
  • 通知账号、学生 council、公告、Credits卡券
  • 投票管理、拍卖审核

  • 智学网认证待审
👤user.html · 用户中心
  • 个人资料展示
  • Credits 流水 & 卡券兑换
  • 每日签到

  • 信任浏览器开关 / 撤销
  • 我的帖子列表
⭐featured.html · 精选
  • 按点赞/评论/时间排序

❓knowledge.html · 问答知识库

🔐 四、认证体系
多账号体系并行
👥普通用户
  • 用户名 + 密码（SHA-256 + 随机盐）
  • Cookie: userId（明文 id）
  • SVG 验证码防注册机
  • 信任浏览器：trustToken + 后端表
  • auto-login 自动恢复会话
🎓智学网账号
  • 可选模块（Playwright 自动化）
  • 绑定流程：pending → approved/denied
  • 管理员 review 后才能用
  • AES-256-CBC 加密存储实名信息
  • 绑定后可直接 zhixue-login
🛠️管理员
  • 独立 admins 表
  • 首位注册者自动 super admin
  • requireAdmin 中间件校验 cookie
  • requireSuper 校验 super===true
  • 超管可添加/删除/修改其他管理员
📢通知发布者
  • 申请 → 管理员审核
  • notice_passkey 密钥登录
  • 独立登录流程（notice.html）
  • 可发通知、可发投票
🎪学生会
  • student_council key-value 表
  • 独立 login / change-pwd / change-name
  • 管理员可重置密码/改名
📱扫码登录
  • /qrcode/generate 生成二维码
  • /scan 扫码
  • /confirm 确认后下发 cookie
  • HMAC-SHA256 token 签名
✨ 五、特色功能
产品级能力
🥸匿名发帖
  • isAnonymous 字段
  • 显示"匿名"+ 🙈头像
  • 隐藏 userId 链接
  • 保留作者 id 供管理员追溯
📊自定义投票
  • allowCustom=true 时允许参与者自定义选项
  • multiple 多选、endTime 截止
  • userId + IP 双重防重复投票
  • 敏感词/霸凌姓名检测自定义内容
🛡️内容安全
  • sensitiveWords.js 敏感词库
  • bullyingNames.js 霸凌姓名保护
  • 敏感词白名单
  • 命中敏感词可强制确认
  • 管理员面板查看敏感统计
💸Credits 经济
  • 签到获取 credits
  • 竞拍今日精选位消耗 credits
  • 问答悬赏 credits
  • 超管生成 credit_cards 兑换码
  • credit_logs 完整流水
🛡️拍卖审核
  • 用户出价后需管理员审核
  • 通过后内容上今日精选位
  • 可举报拍品内容
  • 审核通过后真正扣除 credits
🧰维护模式
  • 管理员面板一键切换
  • checkMaintenance 全局拦截
  • 非管理员看到 maintenance.html
  • admin.html 仍可访问
💬讨论与话题
  • 帖子中可 #话题 关联讨论
  • 讨论区支持话题列表与评论
  • 发帖可同步到指定讨论
  • discussionId 列关联帖子
📦数据安全
  • SQLite WAL 模式 · 事务写入
  • 密码 SHA-256 + salt
  • 智学网实名 AES-256-CBC 加密
  • Token HMAC-SHA256 签名
  • 备份脚本 backup_server.py
👑 六、用户角色与权限
6 种角色分层
👀访客（未登录）
  • 浏览所有公开页面
  • 注册/登录（智学网/扫码）
  • 提交反馈、欺凌/内容举报
👤普通用户
  • 发帖（含匿名）、评论、点赞
  • 参与投票、讨论、问答
  • 竞拍今日精选位、签到、兑换卡券
  • 申请通知发布者
  • 编辑/删除自己的帖
📢通知发布者
  • 发布/编辑/删除通知
  • 置顶、同步到公告
  • 发起投票（含自定义选项）
🛠️管理员
  • 删帖/评、批量操作
  • 处理举报、封禁用户
  • 管理敏感词/霸凌姓名
  • 管理通知/投票/拍卖/公告
  • 切换维护模式
👑超级管理员
  • 管理员的所有能力
  • 增删改其他管理员
  • 生成 Credits 卡券
  • grant/deduct 用户 credits
  • 查看用户详情（绑定审核）
🚫被封禁用户
  • 登录被拒
  • 发帖/评论被拦截
  • user.status = banned 标记
